茨城新聞・今日の注目
JX金属　1200億円投資　光通信分野の素材増産　磯原・ひたちなか工場　茨城
🗞️ 茨城新聞

JX金属は、光通信分野の素材増産のため、最大1200億円を投資する方針を発表した。

  • JX金属は、非鉄金属大手として、次世代の高速通信に欠かせない素材の増産を計画している。
  • 投資額は、最大1200億円で、今後4年間で投じられる予定である。
  • 生産拠点として、磯原工場(茨城県北茨城市)とひたちなか工場(同県ひたちなか市)が選ばれている。
  • ひたちなか工場には、新拠点を整備する計画で、生産能力を拡大する。
  • 生産能力は、2025年度比で7～10倍に引き上げられる予定である。
  • この投資は、茨城県の経済に大きな影響を与える可能性があり、ひたちなか市政にも関連性がある。
  • JX金属のこの投資は、光通信分野の素材需要の増加に対応するためのもので、茨城県の産業発展に貢献することが期待される。
