災害に便乗した悪質商法にご注意を！

災害に便乗した悪質商法にご注意ください。

  • 災害に便乗した悪質商法とは、被害を受けた人々を標的にした詐欺や高額な契約を強要する商法です。
  • 被害を受けた住宅の補修工事に対応できる近隣の事業者は、住まいるダイヤルや消費者ホットラインにご相談ください。
  • 保険に関することは、保険会社又は代理店にご相談ください。
  • 損害保険会社の連絡先は、日本損害保険協会会員会社連絡先ページや外国損害保険協会会員会社連絡先ページからご確認いただけます。
  • 不審・不安に思ったら、消費者ホットライン又は住まいるダイヤルにご相談ください。
  • 契約書に「クーリング・オフできない」と書いてあったり、クーリング・オフ期間を過ぎているように見えても、契約を解除できる場合があります。
  • 訪問販売による取引は、契約書面を受け取った日から8日間以内であれば、契約解除ができます。
  • 「保険金の手続きをサポートする」などの勧誘に注意し、まずは自身が加入している保険契約の内容を確認し、契約している保険会社や代理店に相談しましょう。
