通報報奨金制度について

通報報奨金制度についての情報です。

  • 茨城県では、不法就労外国人に関する通報報奨金制度を立ち上げました。
  • この制度は、県民の皆さまに事業者に関する情報提供を求め、具体的で根拠のある情報に対しては報奨金を支払います。
  • 提供される情報は、事業者に関するものに限定し、外国人個人に関する情報は一切受け付けません。
  • 匿名による通報も受け付けません。
  • 県では、今後も不法就労対策を強化し、外国人が安心して働くことができる環境づくりを進めてまいります。
  • 担当課は産業戦略部労働政策課外国人適正雇用推進室で、電話番号は029-301-3849です。
  • この制度は、真面目に働いている外国人に対する不当な差別や排斥を防ぐための一つです。
  • 県民や事業者の皆さまにおかれましては、御理解と御協力をお願いいたします。
