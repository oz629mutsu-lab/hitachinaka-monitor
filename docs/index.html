台風6号に伴う暴風雨により倒木の恐れがある一部の公園を一時閉鎖しています

ひたちなか市では、台風6号に伴う暴風雨により倒木の恐れがある一部の公園を一時閉鎖しています。

  • 公園の閉鎖は、安全確保のため行っています。
  • 閉鎖する公園は、市内で倒木の恐れがある場所に限ります。
  • 閉鎖期間は、台風の影響がなくなるまでとします。
  • 市民の安全を守るため、協力してください。

この閉鎖は、市民の安全を守るための措置であり、ひたちなか市議会も市民の安全を第一に考えています。

ひたちなか市立学校の教育職員に関する業務量管理・健康確保措置実施計画

ひたちなか市立学校の教育職員に関する業務量管理・健康確保措置実施計画についての情報です。

  • 教育職員の業務量を適切に管理し、健康の確保を図るための計画を定めることとされています。
  • 計画の期間は令和８年から令和１１年までとされています。
  • 実施する業務量管理・健康確保措置の内容として、業務の３分類を踏まえた業務の見直しや、保護者等からの過剰な苦情や不当な要求等の対応などが含まれています。
  • 目標値として、時間外在校等時間が４５時間以下の人数を増やし、時間外在校等時間の平均を３０時間程度に抑えることが掲げられています。
