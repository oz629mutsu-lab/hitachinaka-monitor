通報報奨金制度について

通報報奨金制度についての情報です。

  • 通報報奨金制度は、不法就労外国人に関する通報を行った場合に報奨金を支払う制度です。
  • この制度は、不法就労が多い状況を改善し、法に基づき適正に働いている外国人への差別を防ぐ目的で設けられました。
  • 提供される情報は、事業者に関するものに限定され、外国人個人に関する情報は一切受け付けません。
  • 匿名による通報も受け付けません。
  • 提供された情報は、個人情報保護等に配慮しつつ、情報提供者への聴取や事業者への個別訪問などを通じて確認を行った上で県警に提供されます。
  • この制度は、2026年4月2日に茨城県で開始されました。
  • 担当課は産業戦略部労働政策課外国人適正雇用推進室です。
  • 電話番号は029-301-3849です。

この制度は、不法就労対策を強化し、外国人が安心して働くことができる環境づくりを進めるためのものです。
