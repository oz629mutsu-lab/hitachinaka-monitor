📌 茨城県 注目情報

通報報奨金制度について

通報報奨金制度についての情報です。

  • 通報報奨金制度は、不法就労防止に向けて、県民の皆さまに事業者に関する情報提供を求めるものです。
  • 提供を受ける情報は、事業者に関するものに限定し、外国人個人に関する情報は一切受け付けません。
  • 匿名による通報も受け付けません。
  • 提供された情報は、個人情報保護等に配慮しつつ、情報提供者への聴取や事業者への個別訪問などを通じて確認を行った上で県警に提供することになります。
  • この制度は、犯罪やルール違反の温床となっている不法就労者を受け入れる事業者による違法行為を未然に防ぐために、広く県民の皆さまに情報提供をお願いするものです。
  • 県では、今後も不法就労対策を強化し、外国人が安心して働くことができる環境づくりを進めてまいります。
  • 担当課は産業戦略部労働政策課外国人適正雇用推進室で、電話番号は029-301-3849です。

救急搬送における選定療養費の徴収開始について

救急搬送における選定療養費の徴収についての情報です。

  • 救急搬送における選定療養費の徴収は、救急車で搬送された方のうち、救急車要請時の緊急性が認められない場合に、対象病院において徴収されます。
  • 対象病院は23病院で、初診時に紹介状なしで上記病院を受診すると選定療養費の支払いが必要となります。
  • 選定療養費の徴収は、救急車を有料化するものではなく、既存の選定療養費制度の運用を見直したものです。
  • 選定療養費は、対象病院の会計時に診察料などと一緒に、患者本人からお支払いいただきます。
  • 救急車要請時の緊急性が認められない可能性がある主な事例としては、軽い切り傷のみ、軽い擦り傷のみ、緊急性が低い症状などがあります。
  • 救急車を呼ぶべきか判断に迷う場合は、「茨城県救急電話相談」へご相談ください。
