通報報奨金制度について

茨城県が不法就労外国人に関する通報報奨金制度を導入したことについての情報です。

  • 茨城県は、2026年4月2日に「不法就労外国人に関する通報報奨金制度」を立ち上げました。この制度は、加速度的な人口減少による人手不足が深刻化する中で、意欲と能力のある外国人材を積極的に受け入れ、本県の経済成長を持続させることを目的としています。
  • 制度導入の背景には、茨城県の不法就労者数が3年連続で全国最多であること、直近5年間（2021年～2025年）における外国人の摘発人数が全国で4割減少する一方で、本県では45％増加している現状があります。県民からは不法就労が治安悪化の温床となっているとの不安の声も寄せられています。
  • これまでの不法就労防止策として、事業者自ら外国人材の適正雇用を宣言する制度の創設や、事業者への個別訪問による啓発・調査に取り組んできましたが、その成果は限定的でした。また、県職員による在留資格確認も法律上の制約により実施困難と判断されました。
  • 本制度は、犯罪やルール違反の温床となっている不法就労者を受け入れる事業者による違法行為を未然に防ぐため、県民からの事業者に関する積極的な情報提供を求めるものです。具体的で根拠のある有益な情報に対しては、謝礼として報奨金が支払われます。
  • 情報提供の対象は事業者に関するものに限定され、外国人個人に関する情報は一切受け付けません。また、匿名による通報も受け付けず、提供された情報は個人情報保護に配慮しつつ、情報提供者への聴取や事業者への個別訪問を通じて確認を行った上で茨城県警察に提供されます。
  • 茨城県は、「不法就労外国人に関する通報報奨金制度に反対する会長声明」等の一部批判に対し、排外主義の助長や国籍・人種等を理由とした外国人の差別にはつながらないとの見解を示しています。むしろ、不法就労を放置することが、法に基づき適正に働く外国人への人権侵害や地域経済への大きな打撃になり得ると主張しています。
  • この制度は、茨城県が独自に不法就労対策を強化する姿勢を示しており、他の地方自治体にも同様の制度導入

参政党とチームみらいが企業・団体献金の見直しに向けた禁止法案を共同で衆議院に提出した情報です。

  • 計画期間は令和8年度を初年度とする今後5年間で、整備完了延長約1,000kmの目標を設定し、併せて約4,000kmの計画を策定します。
  • 本計画の3つの主要ポイントは以下の通りです。第一に、災害時の道路啓開の実効性を早期に確保するため、高速道路インターチェンジと県庁等を結ぶ区間を優先整備区間として選定し、重点的に整備を進めます。中長期的には、これらの区間について今後30年間で無電柱化を概ね完成させることを目指します。
  • 第二に、児童の事故リスクを着実に低減するため、通学路を無電柱化の対象とした目標値を新たに設定し、子どもの安全確保を強化します。
  • 第三に、地域全体で切れ目のない景観を創出するため、観光地等で面的な無電柱化を図り、市町村における計画策定と景観・観光部局等との連携を強化します。
  • 地方自治体への影響として、無電柱化工事には多額の費用がかかるため、国からの補助金や交付金の確保が重要となります。また、計画策定や工事実施において、住民合意形成や関係機関との調整が求められます。
